# VantUI  Uniapp 示例

本示例原本是从vant官方发布的微信演示示例转换而来,支持微信小程序（H5不支持！！）。 vant-weapp版本为1.6.7.

vant weapp的组件库，都在工程的wxcomponents目录下。pages目录下是各个示例页面。






## 图片上传服务
提供了图片Nodejs服务 ，代码路径为\src\node-image-upload-server

安装npm包

```
npm install
```
运行服务
```
node app
```
